# RaumMotor High Level Design Document

This document contains the highest level of the game engine design and is used as the main guide while developing the actual engine.

## Core Features

RaumMotor is intended as the next iteration of the T3Vtech-3 game engine. It can be found be founnd [here](https://github.com/tevoran/T3Vtech-3).
The biggest design flaw of T3Vtech-3 was its inflexibility.
